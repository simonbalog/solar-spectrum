# Mellotech — Student Rocket Development Team

Webová prezentace studentského raketového týmu Mellotech. Postaveno s [Astro](https://astro.build).

## 🚀 Struktura

```text
/
├── public/
├── src/
│   ├── components/
│   ├── layouts/
│   ├── pages/
│   └── styles/
└── package.json
```

## 🧞 Příkazy

| Příkaz                    | Akce                              |
| :------------------------ | :-------------------------------- |
| `npm install`             | Instalace závislostí              |
| `npm run dev`             | Lokální dev server na :4321       |
| `npm run build`           | Build do `./dist/`                |
| `npm run preview`         | Náhled buildu lokálně             |
